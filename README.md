# friendly-waddle 🌅

**אפליקציית כושר אישית** (PWA לאייפון) סביב **"הרחבת ידע בוקר"** —
תדריך ידע יומי מבוסס-ראיות שנוצר אוטומטית כל בוקר, יחד עם מעקב אימונים,
יומן אוכל ושקילה.

## מה יש באפליקציה (`app/`)
- 🌅 **תדריך בוקר** — תדריך יומי בעברית (תוספים, אימונים, תזונה) + ארכיון
- 💪 **אימון** — מעקב Progressive Overload עם הצעת התקדמות אוטומטית
- 🥗 **אוכל** — יומן אכילה עם מאגר מאכלים, קלוריות/חלבון ויעדים יומיים
- ⚖️ **שקילה** — רישום משקל + גרף התקדמות
- ⚙️ **עוד** — תזכורת יומית ל-iOS, גיבוי/שחזור, ערכת נושא

## איך זה עובד
1. **שגרה (Routine)** של Claude Code רצה כל בוקר ומריצה את `daily-briefing-prompt.md`.
2. היא חוקרת מקורות עדכניים ושומרת תדריך חדש ב-`briefings/<תאריך>.md`.
3. **אפליקציית ה-PWA** (`app/`) מציגה את התדריכים ומנהלת את המעקב האישי במכשיר.

## קבצים
- `daily-briefing-prompt.md` — ההוראות שהשגרה מריצה כל בוקר.
- `briefings/` — התדריכים שנשמרים (`<תאריך>.md`) + `index.json`.
- `app/` — אפליקציית האייפון (PWA). ראו `app/README.md`.
- `SETUP.md` — מדריך הקמה: השגרה, GitHub Pages, והוספה למסך הבית.

## התחלה מהירה
ראו **`SETUP.md`** — מסביר צעד-אחר-צעד איך להפעיל את השגרה היומית,
לפרסם את האפליקציה דרך GitHub Pages, ולהוסיף אותה למסך הבית באייפון.
