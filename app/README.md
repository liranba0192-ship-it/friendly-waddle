# 📱 אפליקציית "הרחבת ידע בוקר" (PWA)

אפליקציית כושר אישית קלה (PWA, ללא שרת) עם 5 חלקים. אפשר "להוסיף למסך הבית"
באייפון והיא מתנהגת כמו אפליקציה מקורית — אייקון, מסך מלא, ועבודה אופליין.

## תכונות
| טאב | מה הוא עושה |
|-----|-------------|
| 🌅 **בוקר** | מציג את התדריכים היומיים מתוך `../briefings/` (היומי + ארכיון) |
| 💪 **אימון** | מעקב Progressive Overload: רישום סטים (משקל×חזרות), היסטוריה, והצעת התקדמות אוטומטית |
| 🥗 **אוכל** | יומן אכילה עם מאגר מאכלים בעברית, חישוב קלוריות/חלבון, סיכום יומי מול יעדים |
| ⚖️ **שקילה** | רישום משקל גוף + גרף התקדמות (SVG) + משקל יעד |
| ⚙️ **עוד** | תזכורת יומית (קובץ `.ics` ל-iOS), גיבוי/שחזור נתונים, ובחירת ערכת נושא |

## נתונים ופרטיות
כל הנתונים נשמרים **מקומית במכשיר** (`localStorage`) — פרטי, מהיר, ועובד אופליין.
המידע אינו מסונכרן בין מכשירים; השתמש ב**גיבוי לקובץ** (טאב "עוד") כדי לשמור עותק.

## מבנה קבצים
| נתיב | תפקיד |
|------|-------|
| `index.html` | מבנה + ה-meta של ה-PWA (תמיכת iOS) |
| `styles.css` | עיצוב כהה/בהיר אוטומטי, RTL, טאבים |
| `app.js` | ניווט בין טאבים + ערכת נושא |
| `js/store.js` | אחסון מקומי, עזרי תאריך, גיבוי/שחזור |
| `js/briefing.js` · `workout.js` · `food.js` · `weight.js` · `more.js` | המודולים של כל טאב |
| `data/foods.json` | מאגר המאכלים (ערכים ל-100 ג', מקורבים) |
| `vendor/marked.min.js` | המרת Markdown ל-HTML (מקומי, ללא CDN) |
| `icons/` | אייקוני האפליקציה (`tools/gen-icons.mjs`) |

## הרצה מקומית (לבדיקה)
מהשורש של ה-repo:
```bash
python3 -m http.server 8000
# דפדפו אל: http://localhost:8000/app/
```
חשוב להריץ מהשורש (לא מתוך `app/`), כי האפליקציה קוראת את `../briefings/`.
