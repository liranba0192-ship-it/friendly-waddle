# הגדרת עדכון הבוקר האוטומטי ⏰

מדריך פשוט להפעלת **עדכון בוקר יומי** (תוספים, אימונים, תזונה) באמצעות
**Routines** של Claude Code — עם **התראה לאפליקציה בנייד**.

## למה צריך את זה?
סביבת Claude Code on the web היא זמנית ונסגרת כשהשיחה מסתיימת.
"Routine" (שגרה מתוזמנת) רצה על תשתית הענן של Anthropic כל בוקר באופן עצמאי,
מריצה את ההוראות שבקובץ `daily-briefing-prompt.md`, ומכינה את הבריפינג.

## חלק א' — יצירת השגרה (מהדפדפן)

1. פתח: **https://claude.ai/code/routines**
2. לחץ **`New routine`**.
3. **שם:** למשל `עדכון בוקר - תוספים אימונים ותזונה`.
4. **Prompt / Instructions** — הדבק בדיוק:
   ```
   קרא את הקובץ daily-briefing-prompt.md מהריפו ובצע את ההוראות שבו במלואן.
   ```
5. **Repositories:** בחר את `friendly-waddle`.
6. **Environment:** השאר `Default` (יש לו גישת אינטרנט לחיפושים).
7. **Select a trigger → Schedule:**
   - תדירות: `Daily`
   - שעה: לבחירתך (למשל 07:00) — לפי אזור הזמן שלך, מתורגם אוטומטית.
8. לחץ **`Create`**. ✅

> בדיקה מיידית: בעמוד השגרה לחץ **`Run now`** כדי להריץ ניסיון עכשיו.
> השהיה: מתג **`Repeats`** מכבה/מדליק את התזמון.

## חלק ב' — קבלת התראה בנייד 📱

1. התקן את **אפליקציית Claude** (App Store / Google Play) והתחבר לאותו חשבון.
2. אפשר **Notifications** לאפליקציה.

מעכשיו, בכל בוקר תקבל התראה כשההרצה מסתיימת. לחיצה על ההתראה
פותחת את הסשן עם הבריפינג המלא.

## חלק ג' — אפליקציית האייפון 📱 (PWA)

יש בריפו אפליקציית web (PWA) בתיקייה `app/` שמציגה את כל התדריכים יפה,
עם תמיכה מלאה בעברית, ארכיון, ועבודה אופליין. כך מפעילים אותה:

### 1. הפעלת GitHub Pages (פעם אחת)
1. ב-GitHub: **Settings → Pages**.
2. תחת **Build and deployment → Source** בחר **Deploy from a branch**.
3. **Branch:** `main`, **Folder:** `/ (root)`. לחץ **Save**.
4. אחרי דקה־שתיים האתר יהיה זמין בכתובת:
   `https://<שם-המשתמש>.github.io/friendly-waddle/app/`

### 2. הוספה למסך הבית באייפון
1. פתח את הכתובת למעלה ב-**Safari** (חשוב — לא בדפדפן אחר).
2. לחץ על כפתור **שיתוף** ⎙ → **«הוסף למסך הבית»**.
3. עכשיו יש לך אייקון אפליקציה אמיתי 🌅 שנפתח במסך מלא.

> כל בוקר השגרה שומרת תדריך חדש ל-`briefings/`, GitHub Pages מתעדכן אוטומטית,
> והאפליקציה מציגה את התדריך החדש (עם תג "היום"). אין צורך לעדכן ידנית.

## עדכון ידני (מתי שרוצים, בלי להמתין לבוקר)
פתח שיחה ובקש:
> "תן לי את עדכון התוספים, האימונים והתזונה של היום"

## חלק ד' — שגרות טאב הלימוד 📚 (אנגלית + פיננסים)

טאב "לימוד" באפליקציה כולל אוצר מילים לבגרות ומסלול פיננסי. רוב התוכן **קבוע ומסתובב לבד לפי תאריך** (אין צורך בשגרה). שתי שגרות נוספות שומרות על תוכן **טרי וגדל**:

### שגרה 1 — קריאת בוקר באנגלית (יומי) 📖
- **Prompt:** `קרא את הקובץ daily-reading-prompt.md מהריפו ובצע את ההוראות שבו במלואן.`
- **Repositories:** `friendly-waddle` · **Environment:** Default · **Trigger:** `Daily` (למשל 06:30).
- מה היא עושה: יוצרת קטע קריאה אחד באנגלית (~5 דק', רמת בגרות) עם מילון ושאלות, שומרת ל-`readings/` ודוחפת ל-`main`. מופיע בטאב לימוד → אנגלית ככרטיס "קריאת הבוקר".

### שגרה 2 — שיעור פיננסי חדש (יומי) 💰
- **Prompt:** `קרא את הקובץ daily-finance-prompt.md מהריפו ובצע את ההוראות שבו במלואן.`
- **Repositories:** `friendly-waddle` · **Environment:** Default · **Trigger:** `Daily` (שעה לבחירתך).
- מה היא עושה: מוסיפה שיעור פיננסי מתקדם חדש ל-`app/data/finance.json` ודוחפת ל-`main` — כך המסלול גדל ולא חוזר על עצמו.

### שגרה 3 — שיעור AI חדש לפי החדשות (יומי/שבועי) 🤖
- **Prompt:** `קרא את הקובץ ai-guide-prompt.md מהריפו ובצע את ההוראות שבו במלואן.`
- **Repositories:** `friendly-waddle` · **Environment:** Default · **Trigger:** `Daily` או `Weekly` (לבחירתך).
- מה היא עושה: מוסיפה שיעור AI חדש (רמת "להמשך החיים") על ההתפתחות העדכנית בתחום ל-`app/data/ai-guide.json` ודוחפת ל-`main` — כך מדריך ה-AI מתעדכן עם העתיד.

> שלושתן מוגדרות בדיוק כמו שגרת הבוקר (חלק א'), רק עם ה-Prompt והתדירות שלמעלה.

## קבצים בריפו
- `daily-briefing-prompt.md` — הוראות שגרת תדריך הבוקר (תוספים/אימון/תזונה).
- `daily-reading-prompt.md` — הוראות שגרת קריאת הבוקר באנגלית.
- `daily-finance-prompt.md` — הוראות שגרת השיעור הפיננסי היומי.
- `ai-guide-prompt.md` — הוראות שגרת שיעור ה-AI החדש לפי החדשות.
- `SETUP.md` — המדריך הזה.
- `app/` — אפליקציית ה-PWA (האייפון).
- `briefings/` — תדריכי הבוקר (`<תאריך>.md`) + `index.json`.
- `readings/` — קריאות הבוקר באנגלית (`<תאריך>.md`) + `index.json`.
- `app/data/finance.json` — 100 שיעורי המסלול הפיננסי ב-5 רמות (גדל מדי יום).
- `app/data/ai-guide.json` — 100 שיעורי מדריך ה-AI ב-3 רמות (גדל דרך שגרה 3).
- `app/data/vocab.json` — 886 מילים לבגרות (קבוע).
