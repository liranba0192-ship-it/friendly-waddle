# הגדרת עדכון הבוקר האוטומטי ⏰

מדריך קצר לחיבור **טריגר מתוזמן (Scheduled Trigger)** שישלח לך כל בוקר
עדכון ספורט ותזונה, באמצעות Claude Code on the web.

## למה צריך טריגר?
סביבת Claude Code on the web היא זמנית — היא נסגרת כשהשיחה מסתיימת.
כדי לקבל עדכון **כל בוקר באמת**, צריך תזמון חיצוני שמפעיל סשן חדש כל יום.
הטריגר עושה בדיוק את זה.

## שלבים להגדרה

1. היכנס ל-[Claude Code on the web](https://claude.ai/code).
2. בחר את ה-environment המחובר לריפו הזה (`friendly-waddle`).
3. עבור להגדרות ה-**Triggers** ובחר **Scheduled Trigger** (טריגר מתוזמן).
4. הגדר:
   - **תדירות:** יומי (Daily)
   - **שעה:** השעה שבה תרצה לקבל את העדכון (למשל 07:00 לפי אזור הזמן שלך)
   - **Branch:** `claude/daily-sports-nutrition-news-0HpgY` (או `main` לאחר מיזוג)
   - **Prompt:** העתק את התוכן של `daily-briefing-prompt.md`, או פשוט כתוב:
     > קרא את `daily-briefing-prompt.md` ובצע את ההוראות שבו.
5. שמור. מעכשיו תקבל עדכון בכל בוקר בשעה שהגדרת.
