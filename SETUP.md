# הגדרת עדכון הבוקר האוטומטי ⏰

מדריך פשוט להפעלת **עדכון בוקר יומי** (תוספים, אימונים, תזונה) באמצעות
**Routines** של Claude Code — עם **התראה לאפליקציה בנייד**.

## למה צריך את זה?
סביבת Claude Code on the web היא זמנית ונסגרת כשהשיחה מסתיימת.
"Routine" (שגרה מתוזמנת) רצה על תשתית הענן של Anthropic כל בוקר באופן עצמאי,
מריצה את ההוראות שבקובץ `daily-briefing-prompt.md`, ומכינה את הבריפינג.

## חלק א' — יצירת השגרה (מהדפדפן)

1. פתח: **https://claude.ai/code/routines**
2. לחץ **`New routine`**.
3. **שם:** למשל `עדכון בוקר - תוספים אימונים ותזונה`.
4. **Prompt / Instructions** — הדבק בדיוק:
   ```
   קרא את הקובץ daily-briefing-prompt.md מהריפו ובצע את ההוראות שבו במלואן.
   ```
5. **Repositories:** בחר את `friendly-waddle`.
6. **Environment:** השאר `Default` (יש לו גישת אינטרנט לחיפושים).
7. **Select a trigger → Schedule:**
   - תדירות: `Daily`
   - שעה: לבחירתך (למשל 07:00) — לפי אזור הזמן שלך, מתורגם אוטומטית.
8. לחץ **`Create`**. ✅

> בדיקה מיידית: בעמוד השגרה לחץ **`Run now`** כדי להריץ ניסיון עכשיו.
> השהיה: מתג **`Repeats`** מכבה/מדליק את התזמון.

## חלק ב' — קבלת התראה בנייד 📱

1. התקן את **אפליקציית Claude** (App Store / Google Play) והתחבר לאותו חשבון.
2. אפשר **Notifications** לאפליקציה.

מעכשיו, בכל בוקר תקבל התראה כשההרצה מסתיימת. לחיצה על ההתראה
פותחת את הסשן עם הבריפינג המלא.

## עדכון ידני (מתי שרוצים, בלי להמתין לבוקר)
פתח שיחה ובקש:
> "תן לי את עדכון התוספים, האימונים והתזונה של היום"

## קבצים בריפו
- `daily-briefing-prompt.md` — ההוראות שהשגרה מריצה כל בוקר (כולל כללי-ברזל נגד מידע לא מבוסס).
- `SETUP.md` — המדריך הזה.
